# Теория формальных языков и компиляторов

Курсовая работа / Лабораторные работы по дисциплине "Теория формальных языков и компиляторов".

## Лабораторная работа №1: Разработка пользовательского интерфейса (GUI) для языкового процессора

### Цель работы
Создание кроссплатформенного графического интерфейса (GUI) для языкового процессора в виде специализированного текстового редактора.

**Автор:** Шипулин Семен, группа: АВТ-314

---

### Описание проекта
Данное приложение является пользовательским интерфейсом (GUI) для языкового процессора. Оно создано с помощью Windows Forms в Microsoft Visual Studio с использованием языка программирования C#.

Интерфейс программы включает все требуемые компоненты:
- Главное меню с разделами "Открыть", "Правка", "Текст", "Пуск", "Справка"
- Панель инструментов с кнопками быстрого доступа
- Область вывода результатов (DataGridView)
- Область ввода/редактирования текста (RichTextBox)

### Используемые технологии
| Компонент             | Технология                     |
|-----------------------|--------------------------------|
| Язык программирования | C#                             |
| Фреймворк для GUI     | .NET Windows Forms             |
| Среда разработки      | Microsoft Visual Studio 2019   |
| Версия .NET           | .NET Framework 4.7.2 / .NET 6.0|
| Операционная система  | Windows 10/11                  |

---

### Инструкция по сборке и запуску
1. Открыть решение `MainGUI.sln` в Visual Studio
2. Выбрать конфигурацию сборки **Release**
3. Выполнить сборку проекта: **Сборка** → **Собрать решение** 
4. Готовый исполняемый файл: `MainGUIa/bin/Release/MainGUI.exe`

---

### Описание интерфейса и функций (руководство пользователя)
При запуске приложения отображается главное окно с полем для ввода текста, областью вывода результатов, панелью быстрого доступа и главным меню для управления приложением.

![Главное окно приложения](https://github.com/user-attachments/assets/010c6a18-1863-4266-943f-0b563f1d1ca1)

#### Панель инструментов
1. Кнопка новый документ - создание нового файла
2. Кнопка открыть - открытие существующего документа
3. Кнопка сохранить - сохранение текущих изменений
4. Кнопка отмена (Undo) - отмена последнего действия
5. Кнопка повтор (Redo) - повтор отмененного действия
6. Кнопка копировать - копирование выделенного текста
7. Кнопка вырезать - вырезание выделенного текста
8. Кнопка вставить - вставка текста из буфера обмена
9. Кнопка запуск - запуск синтаксического анализатора
10. Кнопка о программе - информация о приложении
11. Кнопка справка - руководство пользователя

### Ограничения

#### Функциональные ограничения
- Реализован только дизайн интерфейса, логика обработки событий не подключена
- Отсутствует реальная функциональность языкового процессора
- Кнопки панели инструментов не обрабатывают события нажатия

#### Технические ограничения
- Приложение работает только на операционных системах Windows
- Требуется установленный .NET Framework / .NET Runtime
- Отсутствует поддержка горячих клавиш
